RAM kya hai?

Operating System ek program hai jo computer hardware aur user ke beech ka link provide karta hai.

OS ke types

  • Batch Operating System
  • Time Sharing OS
  • Distributed OS
  • Real Time OS
  • Network OS

Application Software user ke kaam ke liye bana hota hai. Jaise: MS Word, Excel, PowerPoint.

MS Word kya hai?

MS Word ek word processing software hai jisme hum documents create aur edit karte hain.

MS Word ke Features

  • Text formatting (Bold, Italic, Underline)
  • Paragraph alignment
  • Header & Footer
  • Tables, Images, Shapes
  • Spell Check

Save & Print

Document ko save karne ke liye Ctrl+S aur print karne ke liye Ctrl+P use karte hain.

MS Excel kya hai?

MS Excel spreadsheet software hai jisme data organize, calculate aur analyze kar sakte hain.

MS Excel ke Features

  • Cells, Rows, Columns
  • Formulas & Functions (SUM, AVERAGE, etc.)
  • Charts & Graphs
  • Data Sorting & Filtering
  • Pivot Tables

MS PowerPoint presentation software hai jisme slides create karke ideas visually dikhaye jaate hain.

MS PowerPoint ke Features

World Wide Web (WWW)

WWW ek system hai jisme web pages ek doosre se linked hote hain. Websites, links aur pages iske part hain.

Search Engine kya hai?

Search engine internet par information search karne ke liye tool hai. Jaise: Google, Bing, Yahoo.

  • To ‚Äì Receiver ka address
  • CC ‚Äì Carbon Copy, extra receivers

Python kya hai?

Python ek high-level, interpreted programming language hai. Ye simple aur easy syntax ke liye famous hai.

Python ke Features

  • Simple aur easy to learn
  • Interpreted language
  • Free & open-source
  • Cross-platform
  • Supports object-oriented programming

Python Syntax

Python me code simple aur readable hota hai. Indentation ka use block define karne ke liye hota hai.

Python Example

# Python program to print hello
print("Hello World")

Variables in Python

Variables memory me data store karte hain. Example: 

name = "Gudda"
age = 16

Data Types

  • int ‚Äì integers, jaise 10, 25
  • float ‚Äì decimal numbers, jaise 3.14
  • str ‚Äì string, jaise "Hello"
  • bool ‚Äì True / False

Basic Input/Output

Input lene ke liye input(), aur output ke liye print() use hota hai. 

name = input("Enter your name: ")
print("Hello", name)

Operators

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • % : Modulus

Conditional Statement

age = int(input("Enter age: "))
if age >= 18:
    print("You can vote")
else:
    print("You cannot vote")

Loops

Repeat kaam karne ke liye loops ka use hota hai. Example:

    print(i)

Cyber Safety kya hai?

Cyber Safety ka matlab hai internet ka safe aur responsible use. Hum apni personal info aur data ko protect karte hain.

Cyber Threats

  • Virus ‚Äì Computer ko damage karne wala program
  • Malware ‚Äì Harmful software
  • Phishing ‚Äì Fake websites/email se information chura lena
  • Hacking ‚Äì Unauthorized access to computer/network
  • Spam ‚Äì Unwanted messages/emails

Cyber Security Tips

  • Strong password use karo (letters, numbers, symbols)
  • Personal info kabhi share mat karo
  • Suspicious links ya emails click mat karo
  • Antivirus software use karo
  • Regular updates karo system & apps ke

Safe Social Media Use

  • Privacy settings on rakho
  • Strangers se personal info share mat karo
  • Strong passwords aur 2-step verification use karo
  • Fake profiles aur messages report karo

Important Notes

  • Internet ka responsible use karo
  • Cyber bullying se bachna seekho
  • Backup regularly apne data ka rakho
  • Secure networks use karo, public Wi-Fi avoid karo
